# 🎴 Card UI Design (HTML & CSS)

A simple and clean card design built using **HTML** and **CSS**, demonstrating layout techniques using `position`.

---

## 📌 Project Overview

This project displays a modern **horizontal card layout** that includes:

* Image on the left
* Content (title + description) on the right
* Author info and date aligned at the bottom

The design is inspired by modern UI components used in blogs and product pages.

---

## 🚀 Features

* Clean and minimal UI
* Built using pure HTML & CSS
* Uses `position` for layout control
* Rounded corners and soft shadows
* Organized structure for easy understanding

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Positioning, Box Model, Styling)

## 💡 What I Learned

* How to use `position: relative` and `absolute`

## ⚠️ Notes

* For production projects, consider using **Flexbox** or **CSS Grid**
